Java Maze Game

The Java maze game was an assignment developed during my second semester at university. The goal of the assignment was to develop either a GUI or a command line based maze game in object oriented java. My project used a command line interface that printed the maze at the start and again after each move to represent changes in the current position. Movement uses the basic wasd keys and once the user has reached the end, the program ends.

Python Shopping List

The python shopping list program is a command line interface program developed functional programming paradigm. The code can store and print different recipes and ingredients needed to be able to cook the recipie. C WORD GUESSER

The aim of this assignment was to build a C program from scratch using vim and Subversion to make a word guessing game. It is a command line interface game that based on a random string of letters allocates points for possible words guessed that match a dictionary either given or from a default text file that is read by the program before the game begins.
